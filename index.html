Random Maze Generator

This script allows you to randomly generate a maze of any size in the Unity game engine with a start and endpoint. It is extremely quick, regardless of the maze's size. This script was made to be used for future game projects since many games benefit from randomly generated dungeons.

Loomian Showdown

This is my most recent project and is inspired from the Roblox game Loomian Legacy and the website pokemonshowdown. The project allows its users to easily build a team without the need for breeding, leveling, and other annoying mechanics normally found in game. It then acts as a battle emulator to allow users to challenge each other in turn-based combat.
